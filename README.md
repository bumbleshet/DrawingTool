# DrawingTool
A requirement for the Computer Science Course

## Installation
##### Prerequesites
  Must have a Keras Backend
##### Installing Depencencies

  Python 2:
  ```
  pip install requirements
  ```
  
  Python 3:
  ```
  pip3 install requirements
  ```
  
  Installing Tkinter:
   ```
   sudo apt-get install python3-tk
   ```
